# Bug Reports

File bug reports here when you discover a defect during testing.

## When to File a Bug

- A test fails consistently (not a flaky test)
- You observe unexpected UI behavior that a test covers
- You find a crash, data loss, or incorrect data display

## How to File

1. Copy `test-docs/bug-report-template.md`
2. Save as `bug-reports/bug-XXX.md` where XXX is the next available number
3. Fill in all sections: environment, steps, expected vs actual result
4. Attach screenshots in `screenshots/` if the bug is visual
5. Reference the TC-ID that found the bug

## Naming

`bug-XXX.md` — e.g., `bug-001.md`, `bug-002.md`
